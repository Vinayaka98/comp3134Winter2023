1) What is the definition of server port?
A server port is a communication endpoint on a computer that listens for incoming network traffic. 
It is identified by a unique number, called the port number, which ranges from 0 to 65535.


2) What is server port range?
The server port range refers to the range of port numbers that a server can use to listen for incoming network traffic. 
The range of port numbers is usually determined by the operating system or network administrator.

3) How many bits does number above represent?
Each port number is represented by 16 bits (2 bytes).
